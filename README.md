# cloudcore_churn
ISYS6020 PoC AI-supported customer churn prediction system for CloudCore Network using Flask, Streamlit, and Zapier.
# CloudCore Churn Prediction

ISYS6020 AI in Business Strategy and Management — Semester 1 2026 . This prototype helps CloudCore Network identify customers at risk of churning by analysing usage drop, support ticket volume, and payment delay. 
Customers are scored Low, Medium, or High risk and surfaced to account managers via a Streamlit dashboard for human review before any retention action is taken.
